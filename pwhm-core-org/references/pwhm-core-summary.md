# pWHM core summary

## Startup and FTA override
- amxrt loads /etc/amx/wld/*.odl, then loads modules.
- mod-whm-* constructor hooks load vendor table.
- wld_nl80211_getVendorTable() -> memcpy FTA -> whm_*_fta_*_init() override.
- FSM init sets GEN_FSM_* dependency bits.

## Core flow
- TR-181 API -> FTA (wifiGen/whm_*) -> hostapd/wpa_supplicant/nl80211 -> driver/NVRAM.
- hostapd/wpa_supplicant config generated by wld_*_cfgFile_createExt().
- Daemon control via wifiGen_*_startDaemon() and /var/run sockets.

## FSM dependency flow
- s_performSync -> s_checkPreDependency -> s_syncVap/Ep -> checkRadDependency.
- EP path uses dependency bits; on completion clears FSM_ComPend.

## Trace principles
- Trace must identify HLAPI read/write, parent event "*", SWLA handler, sync function, and mfn_* calls.
- Output includes HLAPI, FTA handler, Org LLAPI (wifiGen_*), Vendor (whm_*), flow type.
